# 인사이트 및 학습 내용

## 🧠 기술적 통찰
- **중앙 데이터 관리**: `contentData.ts`를 통해 전체 사이트의 텍스트와 설정을 관리하므로, 유지보수 시 이 파일 하나만 수정하면 된다는 점을 마스터에게 안내함.
- **Framer Motion 활용**: 모달창 구현 시 `AnimatePresence`를 사용하여 부드러운 등장/퇴장 효과를 적용함으로써 프리미엄한 사용자 경험(UX)을 제공함.
- **조건부 렌더링**: 메인 페이지의 `quickLinks` 매핑 시, 제목에 따라 `Link`(페이지 이동)와 `button`(모달 팝업)을 구분하여 렌더링하도록 처리함.

## 💡 유의사항
- 카드 클릭 시 단순히 페이지 이동뿐만 아니라 모달을 통한 '미리보기' 기능을 선호함(주보의 경우).
- 담임목사님 성함이나 주소 같은 핵심 정보는 전역적으로 일관되게 유지되어야 함.
